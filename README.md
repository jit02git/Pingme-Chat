# 💬 Pingme-Chat

**PingMe Chat** is a modern real-time chat application built using **React.js** for the frontend and **Node.js** with **Socket.IO** for the backend. It enables users to communicate instantly with a responsive UI and real-time updates.

---

## 🚀 Features

✅ Real-time messaging using Socket.IO  
✅ User authentication (JWT)  
✅ One-to-one and group chat support  
✅ Online/offline status  
✅ Typing indicators  
✅ Responsive UI  
✅ Persistent message storage  
✅ Easy to extend and customize

---

## 🌐 Tech Stack

**Frontend:**  
✔ React.js  
✔ Tailwind CSS / CSS Modules  
✔ Axios  
✔ Socket.IO Client

**Backend:**  
✔ Node.js  
✔ Express.js  
✔ Socket.IO  
✔ MongoDB (Mongoose)  
✔ JWT Authentication
